furrblr.

Description

Depending on the selection, the code uses a GET request and fetches images from two different RESTful APIs: one provides dog images, the other provides cat images.
If an image is bookmarked, a POST request is sent to the JSON Server and stores the information.
In the bookmarks section, images are retrieved from the JSON Server and rendered on the screen. Each image receives a button which sends a DELETE request to the server. Images can also be filtered to disply only images of dogs, cats or both.
Viewed images which are not bookmarked are added to a local array and can be cycled through.
